This project is an interactive Healthcare Analytics Dashboard built using Microsoft Power BI.
The dashboard provides a complete overview of hospital operations including patient workflow, bed occupancy, billing insights, doctor feedback, diagnosis trends, and insurance comparison.

It helps hospital management make data-driven decisions by monitoring performance KPIs and financial metrics in real time.
